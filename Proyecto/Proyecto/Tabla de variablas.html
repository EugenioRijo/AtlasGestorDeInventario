Archivo Variable Definición
app.py app Instancia de la aplicación Flask que se utiliza para manejar las rutas y la configuración.
app.py db_config Diccionario que contiene la configuración de conexión a la base de datos (host, user, password, database).
app.py usuario Variable que almacena el nombre de usuario ingresado por el usuario en el formulario de inicio de sesión o registro.
app.py contraseña Variable que almacena la contraseña ingresada por el usuario en el formulario de inicio de sesión o registro.
app.py action Variable que indica la acción que se está realizando, ya sea 'login' o 'register'.
app.py message Variable que almacena mensajes de retroalimentación para el usuario, como errores o confirmaciones.
app.py conn Variable que representa la conexión a la base de datos MySQL.
app.py cursor Variable que permite ejecutar comandos SQL en la base de datos a través de la conexión.
app.py result Variable que almacena el resultado de una consulta SQL, como un registro de usuario encontrado.
app.py productos Variable que almacena la lista de productos obtenidos de la base de datos para mostrarlos en la interfaz.
app.py product_name Variable que almacena el nombre del producto ingresado en el formulario de registro de productos.
app.py unit_price Variable que almacena el precio unitario del producto ingresado en el formulario de registro de productos.
app.py quantity Variable que almacena la cantidad del producto ingresado en el formulario de registro de productos.
app.py total_value Variable que almacena el valor total de los productos en la base de datos, calculado como la suma del precio unitario por la cantidad.
app.py session Objeto que almacena información sobre la sesión del usuario, como el nombre de usuario autenticado.
app.py root Variable que representa la ventana principal de la aplicación Tkinter.
app.py obj Instancia de la clase IMS, que representa la interfaz gráfica de la aplicación de gestión de inventario.
app.py icon_title Variable que almacena la imagen del icono que se mostrará en la barra de título de la ventana.
app.py LeftMenu Variable que representa el marco (Frame) que contiene el menú lateral de la interfaz gráfica.
app.py lbl_menulogo Variable que representa la etiqueta que muestra el logo en el menú lateral.
app.py btn_logout Variable que representa el botón de cierre de sesión en la interfaz gráfica.
app.py lbl_clock Variable que representa la etiqueta que muestra la fecha y hora en la interfaz gráfica.
app.py lbl_empleados Variable que representa una etiqueta que muestra el total de empleados en la interfaz gráfica.
app.py lbl_proveedor Variable que representa una etiqueta que muestra el total de proveedores en la interfaz gráfica.
app.py lbl_categoria Variable que representa una etiqueta que muestra el total de categorías en la interfaz gráfica.
app.py lbl_productos Variable que representa una etiqueta que muestra el total de productos en la interfaz gráfica.
app.py lbl_ventas Variable que representa una etiqueta que muestra el total de ventas en la interfaz gráfica.
inventory.py self.root Variable que representa la ventana principal de la aplicación Tkinter.
inventory.py self.MenuLogo Variable que almacena la imagen del logo que se mostrará en el menú lateral.
inventory.py self.lbl_empleados Variable que representa una etiqueta que muestra el total de empleados en la interfaz gráfica.
inventory.py self.lbl_proveedor Variable que representa una etiqueta que muestra el total de proveedores en la interfaz gráfica.
inventory.py self.lbl_categoria Variable que representa una etiqueta que muestra el total de categorías en la interfaz gráfica.
inventory.py self.lbl_productos Variable que representa una etiqueta que muestra el total de productos en la interfaz gráfica.
inventory.py self.lbl_ventas Variable que representa una etiqueta que muestra el total de ventas en la interfaz gráfica.
main.py app Instancia de la aplicación Flask que se utiliza para manejar las rutas y la configuración.
main.py usuario Variable que almacena el nombre de usuario ingresado por el usuario en el formulario de inicio de sesión o registro.
main.py contraseña Variable que almacena la contraseña ingresada por el usuario en el formulario de inicio de sesión o registro.
main.py conn Variable que representa la conexión a la base de datos MySQL.
main.py cursor Variable que permite ejecutar comandos SQL en la base de datos a través de la conexión.
main.py result Variable que almacena el resultado de una consulta SQL, como un registro de usuario encontrado.
main.py threading Módulo que permite crear y manejar hilos en Python, utilizado para ejecutar la interfaz Tkinter en un hilo separado.
main.py root Variable que representa la ventana principal de la aplicación Tkinter.
main.py obj Instancia de la clase IMS, que representa la interfaz gráfica de la aplicación de gestión de inventario.
styles.css * Selector universal que aplica el estilo a todos los elementos del documento.
styles.css body Estilo que define la apariencia del cuerpo del documento, incluyendo el fondo y la disposición.
styles.css h1, h2, p, span Estilos que definen la apariencia de los encabezados y párrafos en el documento.
inventory.css header Estilo que define la apariencia del encabezado de la página, incluyendo el color de fondo y el espaciado.
inventory.css nav Estilo que define la apariencia de la barra de navegación, incluyendo la disposición de los elementos.
inventory.css table, th, td Estilos que definen la apariencia de las tablas, incluyendo bordes y espaciado.
welcome.css footer Estilo que define la apariencia del pie de página, incluyendo el color de fondo y el texto.
categorias.html {{ url_for('logout') }} Función de Flask que genera la URL para la ruta de cierre de sesión.
empleados.html {{ url_for('wallet') }} Función de Flask que genera la URL para la ruta de la billetera.
index.html {{ url_for('login') }} Función de Flask que genera la URL para la ruta de inicio de sesión.
inventory.html {{ total_value }} Variable que se pasa a la plantilla para mostrar el valor total de los productos en la billetera.
register_product.html {{ url_for('save_product') }} Función de Flask que genera la URL para la ruta de guardar un producto.
welcome.html {{ url_for('static', filename='logo.jpeg') }} Función de Flask que genera la URL para acceder a archivos estáticos, como el logo.
